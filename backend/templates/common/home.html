My Cloud Storage - Облачное хранилище

Веб-приложение для хранения и управления файлами

Доступные функции:

  • Регистрация пользователя
  • Вход в систему
  • Административная панель
  • Файловое хранилище (требуется авторизация)

API endpoints:

  • POST /api/auth/register/ - регистрация
  • POST /api/auth/login/ - вход
  • POST /api/auth/logout/ - выход
  • GET /api/auth/users/ - список пользователей (админ)
  • GET,POST /api/storage/ - работа с файлами
